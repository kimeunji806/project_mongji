📌 Day 1: 화면·DB 설계 및 UI 작성

[오전 : 3시간]

① 화면 설계 및 계획: 서비스 전체 흐름을 고려한 화면 구성 설계 및 설계서 PPT 작성
② 테이블 설계: Oracle DB 기반 테이블 구조 정의 및 생성 스크립트 작성
③ 로그인 페이지: 사용자 인증을 위한 로그인 화면 HTML/CSS 구현
④ 상품 목록 페이지(사용자): 사용자가 상품을 조회할 수 있는 목록 화면 구현
⑤ 상품 등록 페이지(관리자): 관리자가 상품을 등록할 수 있는 화면 구현
※ CSS : 기본 스타일 (복잡한 거 말고 그냥 테두리, 버튼 색깔 정도만)

[오후 : 4시간]

① 프로젝트 구조 설계: Node.js 기반 서버 프로젝트 폴더 구조 구성
② 패키지 설치: npm을 활용한 서버 구동에 필요한 라이브러리 설치
③ DB 연동 설정: Node.js와 Oracle DB 연결 환경 구성
④ 서버 실행 확인: 서버 실행 후 정상 동작(Hello World 출력) 여부 확인

📌 Day 2: 화면 구현 및 기능 연결

[오전 : 3시간]

① 화면 구현 및 기능 연결 (계정목록, 상품목록, 상품등록)
② 샘플 데이터 입력: 테스트를 위한 계정 2개, 상품 데이터 약 5개 등록
③ 시퀀스 생성: 기본 키 자동 증가를 위한 시퀀스 생성

[오후 : 4시간]

① 상품 조회 API: 서버에서 상품 데이터를 조회하는 API 구현
② 상품 목록 출력: JavaScript를 이용해 상품 데이터를 화면에 동적으로 출력
③ 로그인 기능 연동: 로그인 기능을 서버와 연결하여 인증 처리
④ 상품 등록 기능: 상품 등록 버튼 클릭 시 등록 기능 동작 구현
